How to do for booking seat?


The booking is carried out by using of following steps:


1. Select on output available flights the flight will have booking seats.


2. Give a click on button labeled as bookseat.


3. After doing that will be displayed a dialog box, asked for you to chose the number of seats for booking. Select the number of seats for booking.


This way, it will be prompted a dialog box indicates the successful of the operation, if possible, and reflecting the number of left seats.
