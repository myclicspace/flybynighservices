How to do for running connection?


The client-side application has two connection modes: non-network mode and network mode:


After starting the application by using a command line as described in README.TXT file will be prompted the following dialog box, asking to you chose the connection mode.
Give a double click on the connection mode chosen.


According the connection mode chosen will be prompted one of the following dialog boxes:

1. Non-network mode:


2. Network mode:


For each one of cases above it necessary specifiy the appropriate configuration parameters.
All the configuration will be persistent between runs of the program being stored
in a file called suncertify.properties which is located in the current working directory.

If it is first time which the program is being runned ( suncertify.properties is not exist, yet), then is prompted default parameters in that dialog box for settup parameter.
You could accept them, if they are appropriate or you could input to other parameters.

After input to the appropriate configuration parameters give a double click on start button.

NOTE: If the first time that you are running the client application will be displayed the following alert message to you:


This ocurrs because the file suncertify.properties containing all configuration parameter is not exist, yet. Click on OK button. Next time, that dialog box wont be displayed to you.
